# Peer SDK & Onboarding
## Node Eligibility Rules
A residential node only joins the active pool if:
1. **Connection:** Device is on unmetered Wi-Fi.
2. **Power:** Battery is above twenty percent OR currently charging.
3. **Idle:** CPU usage is below the defined threshold to ensure no lag for the user.
## Consent UI Flow
1. **The Hook:** "Unlock Premium for Free."
2. **The Education:** "Your idle bandwidth helps verify global prices."
3. **The Privacy Guarantee:** "No access to personal files, photos, or history."
4. **The Activation:** User manually toggles "Enable Sharing" to start.
